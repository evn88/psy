# Пакетный менеджер проекта

В проекте используется только **npm**.

Источник истины для зависимостей — `package-lock.json`. Версия пакетного менеджера закреплена в `package.json` через
поле `packageManager`.

## Команды

Установка зависимостей:

```bash
npm ci
```

Локальная установка или изменение зависимостей:

```bash
npm install
npm install <package-name>
npm uninstall <package-name>
```

Запуск скриптов:

```bash
npm run <script>
```

Для CLI, который уже присутствует в зависимостях проекта, используйте `npm exec --no -- <command>`.

## Ограничения

- Не используйте `pnpm`, `yarn` или `bun` для установки зависимостей и запуска npm-скриптов.
- Не добавляйте `pnpm-lock.yaml`, `yarn.lock`, `bun.lock` или `bun.lockb`.
- Для воспроизводимой установки в CI используйте `npm ci`.
- Скрипт `preinstall` автоматически отклоняет установку через другой пакетный менеджер и наличие его lock-файла.
